# 测盟汇 - 测试技术交流平台

## 项目简介

测盟汇是一个专为"电子质量管理协会计算机软硬件和信息系统质量测评分会"设计的综合性测试技术交流平台。该项目采用前后端分离架构，包含Web端管理系统、微信小程序移动端和Spring Boot后端服务，旨在解决传统会议组织过程中的效率低下、信息散乱等问题，为测试从业者提供便捷的技术交流、学习培训和信息获取平台。

## 项目背景

### 业务痛点
在开发过程中，我们组发现传统的会议组织方式存在以下痛点：

1. **会议注册效率低下**: 基于电子邮件的会议注册方式工作量大、及时性差，参会者信息更新困难，容易在转换过程中操作出错。

2. **信息发布散乱**: 微信群的通知和新闻发布方式不够有序，重要信息容易被其他回复和点赞淹没，会员单位成员难以查阅历届会议的相关信息。

3. **知识传播受限**: 优秀会议报告只能在现场收看，无法在线回放，影响了知识的传播和共享。

### 解决方案
通过开发这套完整的数字化平台，实现了：
- 在线会议注册系统，提高注册效率
- 统一的资讯发布平台，便于信息查阅
- 在线课程回放功能，促进知识传播
- 移动端便捷访问，提升用户体验

### 技术栈详情

#### Web前端 (frontend/)
- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router 4
- **HTTP客户端**: Axios
- **样式预处理**: SCSS

#### 微信小程序 (miniapp/)
- **框架**: 微信小程序原生框架
- **开发语言**: JavaScript + WXML + WXSS
- **状态管理**: 全局数据 + 页面数据
- **网络请求**: wx.request API
- **存储**: wx.setStorageSync/wx.getStorageSync

#### 后端服务 (backend/)
- **框架**: Spring Boot 2.7.x
- **数据库**: MySQL 8.0
- **ORM框架**: MyBatis Plus
- **认证方式**: Session + JWT Token
- **文件存储**: 本地文件系统
- **构建工具**: Maven
- **开发语言**: Java 8+

## 功能模块

### 1. 用户管理与认证系统

#### 权限体系设计
- **超级管理员(role=3)**: 拥有系统所有权限，可以管理所有用户、租户、部门
- **企业管理员(role=2)**: 管理本企业用户和资源，可以创建和管理本企业内容
- **普通管理员(role=1)**: 管理本部门资源，可以创建和管理本部门内容
- **普通用户(role=0)**: 仅可查看和参与，主要用于小程序端用户

#### 核心功能
- **用户认证**: 支持用户名密码登录、微信小程序登录集成
- **用户管理**: 用户注册、信息修改、状态管理、权限分配
- **个人信息管理**: 基本信息修改、密码修改、头像上传
- **租户管理**: 企业信息管理、部门管理、管理员分配

### 2. 会议管理系统

#### Web端功能
- **会议创建**: 支持详细的会议信息录入
- **会议编辑**: 会议信息修改、状态管理
- **会议审核**: 企业用户创建的会议需要超级管理员审核
- **会议浏览**: 支持按日期、名称、创建人等条件过滤
- **会议详情**: 完整的会议信息展示

#### 小程序端功能
- **会议分类浏览**: 会议研讨、标准定制、技术培训、工具研发、公益行动
- **会议列表展示**: 会议封面、名称、时间信息
- **会议详情查看**: 完整的会议信息、议程、嘉宾介绍
- **参会回执填写**: 单位、姓名、联系方式、到达方式等详细信息

### 3. 课程管理系统

#### Web端功能
- **课程信息管理**: 课程名称、封面、简介、视频等
- **视频上传**: 支持大文件上传，带进度显示
- **审核流程**: 企业用户创建的课程需要超级管理员审核
- **课程排序**: 支持自定义排序，优化展示效果
- **批量操作**: 支持批量删除、状态修改等

#### 小程序端功能
- **课程列表展示**: 课程封面、名称、简介、作者信息
- **课程详情查看**: 完整的课程信息、视频播放功能
- **在线报名**: 课程报名表单、报名状态跟踪
- **学习记录**: 已报名课程列表、学习进度跟踪

### 4. 行业动态管理系统

#### Web端功能
- **动态发布**: 支持富文本编辑、图片上传
- **审核机制**: 普通用户发布需要管理员审核
- **内容管理**: 动态列表查询、搜索、分页、状态管理
- **权限分配**: 超级管理员管理所有动态，企业用户管理自己的动态

#### 小程序端功能
- **动态列表展示**: 按时间倒序排列，支持下拉刷新和上拉加载
- **搜索功能**: 支持标题和内容模糊匹配
- **动态详情**: 富文本内容展示、图片和文字混排
- **用户体验**: 加载状态提示、空状态处理、错误信息友好展示

### 5. 数据分析与反馈系统

#### 数据统计
- **会议统计**: 总会议数、参与人次、月度趋势、类型分布
- **课程统计**: 总课程数、报名人次、月度趋势、分类分布
- **用户行为分析**: 用户活跃度、参与度、偏好分析

#### 可视化展示
- **图表展示**: 使用ECharts展示数据趋势
- **报表导出**: 支持数据导出功能
- **实时监控**: 系统运行状态监控

### 6. 个人中心系统

#### Web端功能
- **个人信息管理**: 头像、昵称、联系方式等
- **密码修改**: 安全的密码更新机制
- **系统设置**: 个性化配置选项

#### 小程序端功能
- **用户认证**: 微信快速登录、手机号密码登录
- **个人信息管理**: 头像、昵称、联系方式、性别、邮箱等
- **密码管理**: 密码修改、验证码验证
- **我的课程**: 已报名课程列表、学习进度跟踪
- **意见反馈**: 用户反馈收集、问题类型分类

## API文档

### 基础配置
- 基础URL: http://localhost:8080
- 认证方式: Session + JWT Token
- 请求格式: JSON
- 响应格式: JSON

### 主要接口

#### 用户相关
- `POST /user/login` - 用户登录
- `GET /user/list` - 获取用户列表
- `POST /user/updateUser` - 更新用户信息
- `GET /user/loadMyself` - 获取个人信息

#### 会议相关
- `GET /confer/list` - 获取会议列表
- `POST /confer/add` - 创建会议
- `PUT /confer/update` - 更新会议
- `DELETE /confer/delete` - 删除会议
- `POST /confer/receipt` - 提交参会回执

#### 课程相关
- `GET /course/list` - 获取课程列表
- `POST /course/add` - 创建课程
- `PUT /course/update` - 更新课程
- `DELETE /course/delete` - 删除课程
- `POST /course/enroll` - 课程报名

#### 动态相关
- `GET /article/page` - 获取动态列表
- `POST /article/add` - 发布动态
- `PUT /article/update` - 更新动态
- `DELETE /article/delete` - 删除动态

#### 数据分析相关
- `GET /analytics/comprehensive-report` - 获取综合报告
- `GET /analytics/conference-stats` - 获取会议统计
- `GET /analytics/course-stats` - 获取课程统计

## 常见问题

### 1. 数据库连接失败
- 检查数据库服务是否启动
- 确认数据库连接配置是否正确
- 验证数据库用户权限

### 2. 前端启动失败
- 检查Node.js版本是否符合要求
- 确认依赖安装是否完整
- 验证端口是否被占用

### 3. 小程序接口调用失败
- 检查后端服务是否正常运行
- 确认API地址配置是否正确
- 验证网络连接和防火墙设置

### 4. 文件上传失败
- 检查文件上传目录权限
- 确认文件大小是否超限
- 验证文件格式是否支持
