# 测盟汇微信小程序

## 项目简介

测盟汇微信小程序是我个人独立开发的移动端应用，为"电子质量管理协会计算机软硬件和信息系统质量测评分会"提供便捷的移动端服务。通过微信小程序平台，用户可以随时随地查看行业动态、参与会议、学习课程，实现了PC端管理系统的完美补充。

## 项目背景

在开发Web端管理系统后，我发现用户对移动端访问的需求日益增长。传统的PC端访问方式在移动场景下体验不佳，特别是在会议现场、出差途中等场景下。因此，我决定开发配套的微信小程序，为用户提供更便捷的移动端体验。

## 技术架构

### 前端技术栈
- **框架**: 微信小程序原生框架
- **开发语言**: JavaScript + WXML + WXSS
- **状态管理**: 全局数据 + 页面数据
- **网络请求**: wx.request API
- **存储**: wx.setStorageSync/wx.getStorageSync

### 后端技术栈
- **框架**: Spring Boot 2.x
- **数据库**: MySQL 8.0
- **认证方式**: JWT Token
- **文件存储**: 本地文件系统

## 核心功能模块

### 1. 首页模块 (pages/index/index)

我设计了信息丰富的首页，为用户提供平台概览：

- **轮播图展示**: 展示最新的会议、课程、动态信息
- **平台简介**: 测盟汇平台介绍和特色功能说明
- **同盟会员单位**: 展示合作企业信息，增强平台公信力
- **快速入口**: 提供主要功能的快捷访问

**技术实现**:
- 使用swiper组件实现轮播效果
- 图片懒加载优化性能
- 响应式布局适配不同屏幕尺寸

### 2. 会议管理模块 (pages/meeting/meeting)

这是小程序的核心功能之一，我重点解决了移动端会议参与的问题：

- **会议分类浏览**: 
  - 会议研讨：技术交流会议
  - 标准定制：行业标准制定会议
  - 技术培训：专业技能培训
  - 工具研发：测试工具开发讨论
  - 公益行动：行业公益活动

- **会议列表展示**: 
  - 会议封面、名称、时间信息
  - 支持搜索和筛选功能
  - 分页加载优化性能

- **会议详情查看**:
  - 完整的会议信息展示
  - 会议议程和嘉宾介绍
  - 参会回执填写功能

- **参会回执功能** (pages/meeting/receipt/receipt):
  - 单位、姓名、性别、联系方式
  - 到达方式、车次、时间等详细信息

### 3. 行业动态模块 (pages/trends/trends)

为用户提供最新的行业资讯和技术动态：

- **动态列表展示**: 
  - 按时间倒序排列

### 4. 课程学习模块 (pages/course/course)

为测试从业者提供专业的技术培训资源：

为用户提供个性化的服务和管理功能：

- **用户认证**:

## 技术特色

### 1. 原生小程序开发
我选择使用微信小程序原生框架进行开发，确保最佳的兼容性和性能表现。原生开发避免了跨平台框架可能带来的兼容性问题。

### 2. 模块化架构设计
- **页面模块化**: 每个功能模块独立成页面
- **组件复用**: 公共组件如搜索栏、列表项等
- **工具函数**: 网络请求、数据处理等工具函数封装

### 3. 用户体验优化
- **加载优化**: 骨架屏、加载动画
- **交互反馈**: 操作成功/失败提示
- **错误处理**: 网络异常、数据异常处理
- **离线支持**: 基础功能的离线缓存

### 4. 数据管理
- **本地存储**: 用户信息、登录状态等
- **全局状态**: 应用级别的数据管理
- **页面状态**: 页面级别的数据管理

## 开发环境搭建

### 环境要求
- 微信开发者工具 最新版本
- 小程序基础库版本 2.19.4+
- Node.js 16+ (可选，用于构建工具)

### 开发步骤

1. **安装微信开发者工具**
   - 下载并安装微信开发者工具
   - 注册微信小程序账号

2. **项目配置**
   ```bash
   # 克隆项目
   git clone [项目地址]
   
   # 进入小程序目录
   cd miniapp
   ```

3. **开发者工具配置**
   - 打开微信开发者工具
   - 导入项目目录
   - 配置AppID（在project.config.json中）
   - 设置开发环境

4. **后端服务配置**
   - 确保后端服务运行在 http://localhost:8080
   - 配置API接口地址（在utils/constants.js中）

### 配置说明

**project.config.json**:
```json
{
  "appid": "your_app_id",
  "projectname": "测盟汇小程序",
  "setting": {
    "urlCheck": false,
    "es6": true,
    "enhance": true
  }
}
```

**API配置** (utils/constants.js):
```javascript
const BASE_URL = 'http://localhost:8080';
```

## 常见问题

### 1. 网络请求失败
- 检查后端服务是否正常运行
- 确认API地址配置是否正确
- 检查网络连接和防火墙设置

### 2. 登录失败
- 确认微信登录配置是否正确
- 检查后端登录接口是否正常
- 验证用户信息是否正确

### 3. 图片加载失败
- 检查图片路径是否正确
- 确认图片文件是否存在
- 验证图片格式是否支持
